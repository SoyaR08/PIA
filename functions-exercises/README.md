### Enunciados

FACTORIAL: Escribir una función que calcule el factorial de un número dado.  
 PRIMO: Crear una función que determine si un número es primo. 
 DIGITOS: Escribir una función que sume los dígitos de un número entero positivo. 
 CONVERSIONES: Crear una función que convierta temperaturas de grados Celsius a 
Fahrenheit. 
 SUMAS: Escribir una función que reciba una lista y devuelva la suma de sus elementos. 
 FIBONACCI: Crear una función que genere la serie de Fibonacci hasta un número dado 
de término.  
 MCD: Escribir una función que encuentre el MCD (Máximo Común Divisor) de dos 
números utilizando el algoritmo de Euclides.  
 CONTAR: Generar una función que cuente cuántas vocales hay en una cadena. 
 ORDEN: Crear una función que ordene una lista de números de menor a mayor sin usar 
el método sort()  
 ÁREA: Escribir una función que calcule el área de un círculo dado su radio.  
 TIEMPO: Generar una función que tome un número total de horas, minutos y segundos, 
y devuelva el tiempo total en segundos.  
 TIEMPO SEGUNDOS: Generar una función que tome un número total de segundos y lo 
convierta a horas, minutos y segundos.  
 PASOS: Crear una función que calcule la distancia aproximada recorrida, dado un 
número de pasos, asumiendo que la longitud media de un paso es de 0.78 metros 
(promedio para adultos). Convertir la cantidad de pasos a kilómetros.  

#### Utilizar funciones LAMBDA para los siguientes enunciados:  
 Elevar una función al cuadrado. Utilizar una función lambda para elevar un número al 
cuadrado.  
 Suma de dos números. Crear una función lambda que sume dos números. 
 Revertir una cadena. Utiliza una función lambda para invertir una cadena de texto. 
 Obtener el mayor de dos números. Crear una función lambda que devuelva el mayor 
de dos números.  
 Filtrar números pares en una lista. Usa una función lambda con filter() para obtener 
sólo los números pares de una lista.  
 Verificar si una cadena es palíndromo. Utilizar una función lambda para verificar si una 
cadena es un palíndromo (se lee igual al derecho y al revés)
